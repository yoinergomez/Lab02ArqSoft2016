Super Matricula

Laboratorio #1

Arquitectura de Software

Diego Jose Luis Botia

Siguiendo el proceso del ejercicio de ejemplo, realiza una aplicación con JSF y primefaces para simular el registro de matrícula de un estudiante. Al final del proceso debe aparecer la confirmación de la matrícula del estudiante con un código QR para validar el registro.
